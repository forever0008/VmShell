# 如何取消通过 Amazon 订阅的服务并停止扣费

Amazon 不仅以销售实体商品闻名，还允许用户通过其账户订阅流媒体服务、杂志、应用等多种服务。这种方式虽然方便快速创建新账户，但当需要取消订阅时，您需要通过 Amazon 进行操作。本文将为您详细介绍如何取消通过 Amazon 订阅的服务。

## Amazon 订阅取消步骤

请注意，取消订阅后，您将不再被扣费，但同时也会失去与该订阅相关的所有权益。

### 快速提示：
本指南展示如何通过 Amazon 网站取消订阅。您也可以在移动应用中通过点击右下角的三条线，选择“您的账户”，然后选择“会员资格与订阅”来访问相关页面。

1. **登录并进入账户菜单**  
   在电脑上打开 Amazon 并登录，然后将鼠标悬停在页面右上角的“账户与列表”选项上，以显示下拉菜单。

2. **选择“会员资格与订阅”**  
   点击下拉菜单中的“会员资格与订阅”选项。

   ![如何取消 Amazon 订阅](./images/amazon-subscription-1.jpg)  
   _在任一 Amazon 页面的顶部可以找到“账户与列表”选项。_

3. **管理订阅**  
   在接下来的页面中，您将看到所有激活的 Amazon 订阅。点击右侧的“管理订阅”按钮开始取消流程。

   ![如何取消 Amazon 订阅](./images/amazon-subscription-2.jpg)  
   _您需要点击“管理订阅”以取消。_

4. **关闭自动续订**  
   点击“自动续订”旁边的开关按钮，关闭自动续订以取消订阅。  
   如果某些选项不允许您在此页面关闭自动续订，请按照提供的链接前往另一个 Amazon 会员页面进行取消。

### 取消 Amazon Prime 会员的快捷方式
如果您想取消 Amazon Prime 会员，以下步骤更为快捷：

1. **进入 Prime 会员页面**  
   在“账户与列表”下拉菜单中，选择“Prime 会员”而非“会员资格与订阅”，这将直接带您进入 Prime 面板。

2. **结束会员资格**  
   点击右上角会员资格框中的小箭头，并从下拉窗口中选择“结束会员”。  
   然后在所有出现的页面上点击“取消我的权益”以完成取消。

   ![如何取消 Amazon Prime 订阅](./images/amazon-prime-subscription.jpg)  
   _直接选择 Prime 会员选项将加速取消流程。_
